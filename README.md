Проект - кинопоиск (на минималках).
Сущности:
	User(Пользователь), User_role( 2 роли - зарегистрированный/не зарегистрированный),
	Feedback(отзыв на фильм, может создаваться, дополняться, удаляться), Film, Actor.
Описание:
	Сервис позволяет найти фильм, посмотреть какие актеры в нем играют.
	Пользователи бывают двух типов - с правами администратора и нет,
	Пользователи с правами администратора могут дополнительно добавлять/редактировать/удалять такие 
	сущности как фильм и актеры.
	Обыкновенные пользователи могут лишь оставлять комментарии под фильмами,
	а так же править и удалять их в случае необходимости.
	Структурная схема БД  в файле "Структурная схема БД v1.0.PNG".
